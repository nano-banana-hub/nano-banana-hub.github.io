# Contributing to bananahub-ai.github.io

## Scope

This repo contains the BananaHub gallery, generated machine-readable catalog files, and catalog build scripts.

## Inbound license

Code and original site copy contributed here are accepted under the repo's MIT license.

Sign off every commit with the Developer Certificate of Origin:

```bash
git commit -s -m "feat: your change"
```

## Template-related contributions

If you add a curated template source, moderation rule, or other catalog-facing template change:

- follow [COMMUNITY-TEMPLATE-POLICY.md](./COMMUNITY-TEMPLATE-POLICY.md)
- require explicit template metadata for `author` and `license`
- do not submit third-party sample images, logos, or marketing copy unless the upstream template license allows redistribution
- keep attribution truthful; discovered catalog entries are not relicensed by this repo
