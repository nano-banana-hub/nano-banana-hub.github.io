Quick Start

Install BananaHub Skill before browsing deeper.

Use one install path for your environment, then run setup once and come back to the catalog when you need templates.

Open Agent Skills

npx skills add https://github.com/bananahub-ai/bananahub-skill --skill bananahub

Claude Code

claude skill install https://github.com/bananahub-ai/bananahub-skill

Run setup once

/bananahub init
